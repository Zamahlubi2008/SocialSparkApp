 Social Spark App

 UI AND DESIGN

 The mobile app was builtto provide an easy and attractive interface to the users.
 - Clean layout using LinearLayout
 - Personalized colours for homey "homely"vibes
 - Simple input and output structure

   FUNCTIONALITY

   -User provide input or click a button
   -The app precesses the input
   -It shows a suggestion or response

   TESTING

   The application is tested on Android Emulator. Which differs to ensure all the button work and input functions worked.

   THE USAGE OF GITHUB

   GitHub was used to:
   -Store and manage preject files
   - Track progress
   - Submit the final project files
  
     GITHUB ACTIONS

     To automate the build process we used GitHub Actions. Evertime code is pushed, the project is automatically built to check for errors.
